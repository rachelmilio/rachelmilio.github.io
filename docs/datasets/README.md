## Attic Oratory Database datasets
This repository contains the datasets adapted and presented in the Attic Oratory Database. In the Ontoterminology folder, the four ontoterminologies (Legal Processes, Legal Bodies, Legal Participants, and Oratory Corpus) can be accessed. The Case Studies folder contains the data used for the Factoid Prosopography and Peripleo mapping case studies. This repository also contains the Attic Law Ontology file (as a TTL file), which contains all the legal events, texts, and people in the corpus. Lastly, a sample Python file (person_pages_generator.ipynb) demonstrates how this project generates webpages from the Attic Law Ontology.
