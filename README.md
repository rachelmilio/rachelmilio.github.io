The codebase for the Attic Oratory Database, a static Jekyll site presenting the corpus of Attic Oratory via an event-centric, prosopographical model. The site data is stored in a knowledge graph, the Attic Law Ontology, which contains information on the corpus, legal events, and involved persons. This RDF data is then processed into static HTML via a Python script.
